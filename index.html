Research Projects

Cross-Lingual Speaker Verification

Developed a cross-lingual speaker verification system by leveraging transfer learning. Pre-trained on high-resource English (LibriSpeech) and fine-tuned on Tamil and Malayalam datasets.

Implemented ECAPA-Conformer architecture integrating spectral (Log-Mel, MFCC) and prosodic (pitch, energy) features using multi-head attention fusion.

Tech Stack: PyTorch, Feature Engineering, Representation Learning

Outcome: Improved embedding discrimination and model robustness using hybrid AAM-Softmax + Triplet loss and SpecAugment.

Multilingual Hate Speech Detection

Built an NLP pipeline for multilingual hate speech detection using English and Malayalam social media datasets.

Compared traditional ML models (Decision Tree, Random Forest, XGBoost) with deep learning (CNN + BiLSTM) and transformer architectures (BERT, MuRIL, mBERT, IndicBERT).

Tech Stack: Transformers, Scikit-learn, NLP Preprocessing

Outcome: Addressed class imbalance and optimized hyperparameters to enhance F1-score and generalization.

Indian Food Detection & Nutrient Estimation

Designed a computer vision–based detection system using YOLOv5 to identify Indian food items and associate them with nutritional data.

Conducted optimizer comparisons and threshold tuning experiments to reduce false positives and improve detection accuracy.

Tech Stack: YOLOv5, OpenCV, Deep Learning Optimization

Outcome: Achieved mAP@0.5 of 0.76 with 18% reduction in false positives.

Neonatal HIE Segmentation & Classification

Proposed a two-stage deep learning pipeline combining medical image segmentation with classification for neonatal Hypoxic-Ischemic Encephalopathy analysis.

Integrated feature extraction, model validation, and comparative benchmarking.

Tech Stack: CNN Architectures, Medical Imaging, Evaluation Metrics

Outcome: Dice score 0.749, classification accuracy 89%, macro F1-score 0.83. Presented at IEEE INDICON 2025.
