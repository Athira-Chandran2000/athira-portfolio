Research Projects

Cross-Lingual Speaker Verification

Designed a robust cross-lingual speaker verification system addressing low-resource language challenges in Tamil and Malayalam by leveraging transfer learning from high-resource English datasets (LibriSpeech).

Implemented an advanced ECAPA-Conformer architecture integrating spectral (Log-Mel, MFCC) and prosodic (pitch, energy) features using multi-head attention fusion for enhanced speaker embedding representation.

Enhanced discriminative learning through hybrid AAM-Softmax and Triplet loss. Applied SpecAugment, noise injection, and reverberation augmentation to improve model robustness and cross-language generalization.

Key Contribution: Improved embedding separability and reduced domain gap between high-resource and low-resource languages.

Multilingual Hate Speech Detection

Developed a multilingual NLP pipeline for hate speech detection across English and Malayalam social media datasets.

Conducted comparative experimentation between traditional ML models (Decision Tree, Random Forest, XGBoost), deep learning models (CNN + BiLSTM hybrid), and transformer-based architectures (BERT, MuRIL, mBERT, IndicBERT).

Addressed class imbalance using resampling strategies and optimized hyperparameters using cross-validation for improved macro F1-score.

Impact: Contributed to robust NLP solutions for low-resource languages with strong generalization performance.

Indian Food Detection & Nutrient Estimation

Built a computer vision pipeline to detect Indian food items using YOLOv5 object detection framework and mapped them to nutritional information databases.

Conducted optimizer comparisons, confidence threshold tuning, and bounding-box refinement to reduce false positives and improve detection stability.

Performed systematic evaluation using mAP@0.5 metric and error analysis to understand misclassification patterns.

Outcome: Achieved mAP@0.5 of 0.76 and reduced false positives by 18%, improving real-world applicability.

Neonatal HIE Segmentation & Classification

Proposed a two-stage medical AI framework integrating deep learning–based image segmentation and machine learning classification to analyze neonatal Hypoxic-Ischemic Encephalopathy severity.

Applied feature extraction and comparative benchmarking to validate segmentation and classification consistency.

Evaluated performance using Dice coefficient, classification accuracy, and macro F1-score to ensure reliability across severity grades.

Achievement: Dice score 0.749, accuracy 89%, macro F1-score 0.83. Presented at IEEE INDICON 2025.
