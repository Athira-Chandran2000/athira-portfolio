Projects

Cross-Lingual Speaker Verification for Indic Languages

Designed a robust speaker verification system by pre-training on high-resource English datasets (LibriSpeech) and fine-tuning on low-resource Indic languages like Tamil and Malayalam.

Implemented ECAPA-Conformer architecture integrating spectral features (Log-Mel, MFCC) and prosodic features (pitch, energy) using multi-head attention. Improved embedding discriminability using hybrid AAM-Softmax and Triplet loss.

Impact: Enhanced robustness through SpecAugment and noise augmentation, improving generalization across languages.

Multilingual Hate Speech Detection

Developed a multilingual hate speech detection system for English and Malayalam datasets. Compared traditional ML models (Decision Tree, Random Forest, XGBoost) with deep learning models (CNN + BiLSTM) and transformer-based architectures (BERT, MuRIL, mBERT, IndicBERT).

Addressed class imbalance, optimized hyperparameters, and evaluated performance using precision, recall, F1-score metrics.

Impact: Built a robust NLP pipeline for low-resource languages.

Indian Food Detection & Nutrient Estimation (YOLOv5)

Developed a computer vision pipeline to detect Indian food items and associate them with nutritional information.

Implemented YOLOv5 object detection model and compared optimization strategies.

Result: Achieved mAP@0.5 of 0.76 and reduced false positives by 18% through threshold tuning and optimizer adjustments.

Integrated Segmentation & Classification Framework for Neonatal HIE

Proposed a two-stage deep learning pipeline combining image segmentation and classification for neonatal medical image analysis.

Achieved Dice score of 0.749 and classification accuracy of 89%. Macro F1-score improved to 0.83 ensuring consistency across severity grades.

Achievement: Presented at IEEE INDICON 2025 conference.
